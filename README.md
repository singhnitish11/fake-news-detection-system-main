Fake News Detection System

**Project Overview**

The Fake News Detection System is a Machine Learning-based web application designed to identify whether a news article is Real or Fake. The system utilizes Natural Language Processing (NLP) techniques and a Logistic Regression model to analyze news content and provide accurate predictions.

**Objectives**

- Detect fake and real news articles automatically.
- Reduce the spread of misinformation.
- Provide a simple and user-friendly news verification platform.
- Demonstrate the practical application of Machine Learning and NLP.

**Technologies Used**

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript

**Machine Learning Approach**

- Data Preprocessing
- TF-IDF Vectorization
- Logistic Regression Classification
- Model Evaluation and Prediction

**Features**

- Real-time news classification
- User-friendly web interface
- Fast and reliable predictions
- Machine Learning-based decision making
- Easy deployment and scalability

**How to Run the Project**

1. Clone the repository.
2. Install the required libraries.
3. Run the Flask application:
   python app.py
4. Open the application in your browser.
5. Enter a news article and click "Check News".

**Dataset Information**

The project was trained using Fake News datasets containing real and fake news articles. Due to repository size limitations, dataset files are not included in this repository.

Project Output

The system predicts whether the entered news article is Real News or Fake News based on trained Machine Learning models.
